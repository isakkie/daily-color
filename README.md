# 每日色彩

一个帮助用户每天练习场景化配色的静态网页原型。

当前功能：

- 今日色彩推荐
- 60 组内置场景化配色种子
- 每天首次打开自动推荐一组当日配色
- 根据天气、心情、强度手动刷新推荐
- 穿搭、照片、文档、家居四个生活场景提示
- 上传图片并在本地抽取抽象色块
- 按比例预览综合色彩
- 保存个人色彩模板到浏览器本地
- 导出 / 导入色彩库，用于跨设备迁移

## 色彩库导入 / 导出

- 导出文件类型：`.json`
- 导出内容：浏览器本地保存的色彩模板，不包含原始上传图片
- 导入方式：选择由“每日色彩”导出的 `.json` 文件
- 迁移方式：可以通过 AirDrop、微信、邮件、网盘等方式把 `.json` 文件传到另一台设备后再导入
- 导入策略：合并到当前色彩库，并跳过重复模板，不会直接覆盖已有记录

直接打开 `index.html` 即可使用。
